
Randoop's documentation is in the doc/ directory.

* For the user manual, see doc/index.html

  The user manual includes information on using Randoop to generate
  unit tests.

* For the developer's manual, see doc/dev.html

  The developer's manual includes information on compiling Randoop,
  running Randoop's unit and system tests, administering the project
  page, and an overview of Randoop's internals (userful if you want to
  modify/enhance the tool).

* Javadoc is in doc/javadoc/.
